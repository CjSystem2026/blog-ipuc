# Project Blueprint: Christian Blog (IPUC)

Este documento es el "System Prompt" del proyecto. Define las bases, la visión y las reglas que guiarán todo el desarrollo.

## 1. Visión y Propósito
*   **Misión:** Crear una plataforma digital moderna para la difusión del evangelio y el crecimiento espiritual.
*   **Audiencia:** Miembros de la comunidad, jóvenes buscando respuestas y público en general interesado en valores cristianos.
*   **Tono:** Inspirador, respetuoso, profesional y acogedor.

## 2. Bases del Diseño (Aesthetics)
*   **Concepto:** "Luz y Paz".
*   **Paleta de Colores:** Blancos limpios, azules suaves, maderas cálidas y acentos dorados sutiles.
*   **Tipografía:** Serif para títulos (elegancia/tradición) y Sans-Serif para cuerpo (modernidad/lectura fluida).
*   **Imágenes:** Fotografía de alta calidad, paisajes naturales y momentos comunitarios auténticos.

## 3. Guía Técnica (The Rules)
*   **Framework:** Laravel 11 (Vanguardia).
*   **Infraestructura:** Docker (vía Laravel Sail) para un entorno aislado y profesional.
*   **Arquitectura:** Seguimiento estricto del patrón MVC y Clean Code.
*   **Base de Datos:** MySQL (dentro del contenedor Docker).
*   **Frontend:** Tailwind CSS para un diseño premium y responsive.

## 4. Funcionalidades Clave
*   **Gestión de Artículos:** Editor enriquecido para publicaciones profundas.
*   **Categorías:** Organización temática (Devocionales, Doctrina, Comunidad, etc.).
*   **Versículo del Día:** Un componente dinámico que salude al visitante con la Palabra.
*   **Buscador Optimizado:** Para encontrar temas específicos rápidamente.
